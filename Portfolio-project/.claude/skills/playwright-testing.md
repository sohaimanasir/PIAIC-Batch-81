# Playwright Testing for Portfolio Project

## Purpose

Test my website visually to ensure proper functionality, responsiveness, and UI consistency across different devices and screen sizes.

## What it does

- Take screenshots of key pages and sections of the website
- Check responsive design across common viewport sizes
- Verify UI elements are displaying correctly and functioning as expected
- Identify visual regressions or layout issues

## Key Capabilities

- Capture visual snapshots of the website at different breakpoints
- Test common viewport sizes (mobile, tablet, desktop)
- Verify that UI components render properly
- Check for layout issues or broken elements
- Validate responsive behavior across devices

## Knowledge Base

- Common viewport sizes: 375px (mobile), 768px (tablet), 1024px (desktop), 1440px (large desktop)
- Best practices for responsive design
- UI/UX principles for visual consistency
- Web accessibility standards
- Cross-browser compatibility considerations

## Approach

- Navigate through key pages of the website
- Capture screenshots at different viewport sizes
- Compare visual elements against expected design
- Report any visual inconsistencies or issues found
- Document responsive behavior across devices
